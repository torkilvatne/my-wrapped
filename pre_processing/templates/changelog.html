Wrapped Overview By Year Song Counts Insights Guide Changelog

What's New

A history of updates and improvements to the app.

May 2026 Latest 2026-05-18
New
  • Top Songs by Month — By Year page now shows a month picker below the year selector with top 10 songs for each month and an All Year default view, plus export to Spotify.
  • Changelog page — this page.
Improved
  • Export always sends all 50 songs — Top Songs export on the By Year page no longer depends on whether the list is expanded or collapsed.
Security
  • OAuth CSRF protection — PKCE flow now includes a state parameter; callback rejects mismatched values.
  • Open redirect fix — OAuth return URL is validated to be same-origin before redirecting.
  • Subresource Integrity — Chart.js CDN script now includes an integrity hash to guard against supply-chain tampering.
  • Content Security Policy — Cloudflare Pages _headers file adds CSP, X-Frame-Options, X-Content-Type-Options, and Referrer-Policy.
  • Upload validation — Uploaded JSON is checked for expected structure before being stored.
  • Rate limit safety — Spotify API 429 retry is now iterative with a 30-second cap and 3-retry limit.
May 2026 — Hosted release 2026-05-15
New
  • Cloudflare Pages hosting — the app is now publicly available. Upload spotify_data.json in the browser; no local server needed.
  • IndexedDB persistence — uploaded data persists across page navigations and browser restarts without re-uploading.
  • Browser-based settings — Spotify Client ID is entered in the ⚙ settings panel and stored in localStorage.
  • Guide page — step-by-step instructions for getting your Spotify data and setting up playlist export.
  • Embedded script download — the upload modal includes an expandable section to copy or download generate.py directly.
Fixed
  • Spotify 403 on playlist export — updated to the new /playlists/{id}/items endpoint introduced in February 2026.
Initial release 2026-05-01
New
  • Overview page — all-time stats, top tracks and artists, heatmap, and listening timeline.
  • By Year page — year selector with first track, stats grid, top songs and artists, monthly hours chart, and Spotify export.
  • Song Play Counts page — searchable, sortable table of every track with per-year filtering.
  • Insights page — hourly and daily listening patterns, artist lifecycle chart, platform breakdown, session stats, and country map.
  • Spotify playlist export — PKCE OAuth flow to create playlists directly in your Spotify account.
  • generate.py — zero-dependency Python script that converts Spotify Extended Streaming History into the app's JSON format.
